# Dexpools Trading

## Environment

- Node ^16
- React ^18
- Next ^ 12

## Dependencies

- [Yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable)

## Get Started

### Clone Repository

```
git clone https://github.com/unigisper8/next-material-ui.git
```

### Run

```bash
yarn install
yarn dev
```
